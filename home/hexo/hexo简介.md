# Hexo

## 安装

1. 先安装git和node.js

2. ### 安装 Hexo

   ` npm install -g hexo-cli`

   ` npm install hexo`

3. 可以使用命令

    npx hexo <command> 

4. 初始化

   创建文件夹 例如 `D:\data\hexo`

   进入 `"D:\data\hexo"`打开`git bash here`运行 ` hexo init hexo`

5. 启动

   运行`hexo server`然后浏览器输入` 127.0.0.1:4000 `就可以看到页面了
