硬件设备文件名 : 

| 硬件              | 设备文件名           |
| ----------------- | -------------------- |
| IDE硬盘           | /dev/hd[a-d]         |
| SCSI/SATA/USB硬盘 | /dev/sd[a-p]         |
| 光驱              | /dev/cdrom或/dev/hdc |
| 软盘              | /dev/fd[0-1]         |
| 打印机（25针）    | /dev/1p[0-2]         |
| 打印机（USB）     | /dev/usb/1p[0-15]    |
| 鼠标              | /dev/mouse           |
