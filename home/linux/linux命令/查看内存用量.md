# 查看内存使用

1. 查看当前系统使用总量

```shell
[root@localhost local]# df -h
Filesystem            Size  Used Avail Use% Mounted on
/dev/mapper/vg_admin-lv_root
                       18G   13G  3.6G  79% /
tmpfs                 504M  172K  503M   1% /dev/shm
/dev/sda1             477M   35M  418M   8% /boot
```

2. 查看当前文件夹中文件或者文件夹的使用量

```shell
[root@localhost usr]# du --max-depth=1 -h
4.0K	./games
355M	./bin
4.0K	./etc
43M	./libexec
1.4G	./share
5.2G	./local
437M	./sbin
538M	./java
15M	./include
21M	./sublime_text_3
12K	./src
1.3G	./lib
9.2G	.  -- 总量
```
