```java
  class ArrayAlg{
    public static <T> T getMiddle(T...a){
      return a[a.length/2];
    }
  }
```

泛型方法的调用
```java
  String middle = ArrayAlg.<String>getMiddle("John","Q.","public");

  在大多数情况下,方法调用可以省略<String>类型参数,编译器有足够的信息推断出所调用的方法,也就是说,可以调用 :
  String middle = ArrayAlg.getMiddle("John","Q.","public");

```
