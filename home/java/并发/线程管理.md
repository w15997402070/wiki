## 线程优先级

每个线程都有优先级,范围在`Thread. MIN_PRIORITY`和`MAX_PRIORITY`之间(1到10之间),默认优先级为5

```java
public final static int MIN_PRIORITY = 1;
public final static int NORM_PRIORITY = 5;
public final static int MAX_PRIORITY = 10;
```

优先级范围 : 

```
10 -> 关键问题
7~9 -> 交互事件驱动
4~6 -> IO相关
2~3 -> 后台计算
1 -> 在没有其他程序运行的情况下运行
```
